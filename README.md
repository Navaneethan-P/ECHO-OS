# 🌌 ECHO-OS

**ECHO-OS** is an experimental, gamified "Operating System" interface designed to act as an AI-powered memory and insight engine. By combining beautiful glassmorphism UI with interactive constellation views, ECHO-OS transforms the way you capture thoughts, track progress, and level up your personal knowledge base.

## ✨ Key Features

- 🧠 **Memory Input Engine:** Seamlessly capture thoughts, ideas, and daily logs.
- 🌠 **Constellation View:** Visualize your interconnected memories and insights as a glowing star map.
- 🎮 **Gamified Progression:** Earn experience points (XP) and level up as you interact with your digital brain, complete with custom `LevelUpNotifications`.
- 💎 **Glassmorphism Interface:** A stunning, futuristic `GlassCard` design language that feels like a sci-fi operating system.
- 🤖 **AI Agent Integration:** Automated insights generated from your inputs to find hidden connections in your knowledge base.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Navaneethan-P/ECHO-OS.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd ECHO-OS
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
